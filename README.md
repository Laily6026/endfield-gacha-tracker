# ⬡ Endfield Gacha Tracker

> 명일방주: 엔드필드 가챠 기록 분석 도구

공식 API에서 가챠 기록을 자동 수집하고, 배너별 천장 · 운 점수 · 6성 히스토리를 분석합니다.
CLI 버전(`.py`)과 GUI 버전(`.exe`) 두 가지를 제공합니다.

## ✨ 주요 기능

- 🔍 **자동 토큰 추출** — 게임 로그에서 인증 토큰을 자동으로 읽어 공식 API 호출
- 📡 **전체 기록 수집** — 캐릭터/무기 전 배너 기록을 CSV로 저장
- 📊 **배너별 천장 분석** — 배너를 혼합하지 않고 각각 독립적으로 천장 계산
- 🎰 **운(Luck) 점수** — 이항분포 기반으로 상위 몇 %인지 수치화
- 🖥️ **GUI 지원** — CustomTkinter 기반 다크 테마 UI

## 🚀 실행 방법

### GUI 버전 (권장)

#### 방법 1 — 빌드된 exe 직접 실행
```
EndfieldGachaTracker.exe 더블클릭
```

#### 방법 2 — Python으로 직접 실행
```bash
pip install customtkinter
python endfield_tracker_gui.py
```

#### 방법 3 — exe 직접 빌드
```bash
build_exe.bat 더블클릭
# → dist\EndfieldGachaTracker.exe 생성
```

### CLI 버전
```bash
python endfield_tracker_v1_1.py
```

## 📋 사용 방법

1. **엔드필드 실행** 후 게임 내 **가챠 기록 창**을 한 번 열기
2. 트래커 실행 후 **📡 데이터 수집 시작** 클릭
3. 수집 완료 후 캐릭터/무기 탭에서 결과 확인

> ⚠️ 데이터 수집 전 반드시 게임 내 가챠 기록 창을 열어야 합니다.  
> 로그 파일에 인증 토큰이 기록되어야 API 호출이 가능합니다.

## 🔒 개인정보 안내

- 게임 로그에서 읽은 인증 토큰은 **공식 서버(`ef-webview.gryphline.com`)에만 전송**됩니다.
- 토큰은 **외부 서버로 전송되지 않으며**, 로컬에 저장되지 않습니다.
- 수집된 데이터는 **로컬 CSV 파일**에만 저장됩니다.

## 📊 분석 항목

### 캐릭터 헤드헌팅
| 항목 | 설명 |
|---|---|
| 총 뽑기 횟수 | 전체 캐릭터 가챠 횟수 |
| 6성 획득 수 | 획득한 6성 캐릭터 수 (공식 기대치 비교) |
| 평균 6성 천장 | 여정의 시작 배너 제외 평균 (공식 평균 약 62.5회) |
| 현재 스택 | 마지막 6성 이후 누적 뽑기 수 |
| 운 점수 | 이항분포 기반 상위 % |

### 무기 헤드헌팅
| 항목 | 설명 |
|---|---|
| 총 뽑기 횟수 | 전체 무기 가챠 횟수 |
| 6성 획득 수 | 획득한 6성 무기 수 (공식 기대치 비교) |
| 평균 6성 천장 | 배너별 평균 (공식 평균 약 20.0회) |
| 현재 스택 | 마지막 6성 이후 누적 뽑기 수 |
| 운 점수 | 이항분포 기반 상위 % |

## 🛠️ 업데이트 이력

### v1.1 (현재)
**버그 수정**
- 무기 데이터 2배 중복 수집 제거 (`(타입, seqId)` 복합 키 중복 제거)
- 긴급모집 (`isFree=True`) 천장 계산에서 제외
- 동일 타임스탬프 기록의 정렬 오류 수정 (`gachaTs + seqId` 복합 정렬)
- 캐릭터/무기 판별 오류 수정 (`weaponId` 기반으로 변경)
- `server_id` 하드코딩 제거 (URL에서 동적 추출)

**분석 개선**
- 배너별 독립 천장 계산 (캐릭터/무기 모두)
- 여정의 시작 배너 천장 상한 40회로 수정
- 평균 천장 계산 시 여정의 시작 배너 제외
- 한정 무기고 API 수집 누락 수정

**출력 개선**
- 배너별 6성 히스토리 분리 출력
- 6성 없는 배너도 현재 스택 표시
- 배너 헤더에 총 횟수 / 평균 천장 표시
- 캐릭터 종합 평가 메시지 추가

**GUI 추가**
- CustomTkinter 기반 다크 테마 GUI
- 실시간 프로그레스바
- 캐릭터 / 무기 탭 분리
- 운 점수 게이지 바
- 배너별 히스토리 테이블 (스택 미니 프로그레스바 포함)
- exe 빌드 스크립트 제공
