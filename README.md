# [명일방주: 엔드필드 가챠 분석기]

## [개요]
본 스크립트는 게임 로그를 통해 가챠(헤드헌팅) 기록을 추출하고 확률을 분석하는 도구입니다.
Windows 및 Linux (Steam Proton/Wine) 환경을 모두 지원하며, 외부 라이브러리 설치 없이 Python 내장 모듈만으로 가볍게 동작합니다.
즉, 추가적인 라이브러리 설치가 필요없습니다.

## [새로운 기능 (v1.1)]
- **다중 계정 관리**: 여러 계정(본계/부계)의 가챠 기록을 하나의 DB에 누적 저장할 수 있습니다.
- **웹 뷰어 지원 (`viewer_web.py`)**: 콘솔 대신 깔끔한 웹 브라우저 UI에서 데이터를 필터링하고 분석할 수 있습니다.
- **영구적인 누적 저장**: 기존 CSV 덮어쓰기 단점을 해결하여, `endfield_gacha_history.db` SQLite DB에 모든 뽑기를 영구보존합니다. (중복 뽑기 방지 및 정확한 집계 로직 적용)

## [사용 방법]
1. 시스템에 Python 3.8 이상이 설치되어 있는지 확인합니다. (https://www.python.org)
2. 게임을 실행한 후 게임 내에서 '가챠 기록' 창을 최소 한 번 이상 열어줍니다. (로그 파일에 최신 인증 토큰을 남기기 위함입니다.)
3. 명령어 프롬프트나 터미널에서 스크립트가 위치한 폴더로 이동한 후 아래 명령어를 실행합니다:

### 1-1. 데이터 추출 및 누적 저장 (단일 계정)
```bash
python endfield_tracker_v1.1.py
```

### 1-2. 다중 계정 태그 추가하여 저장 (본계/부계 등)
명령어 뒤에 `[UID]`와 `[별명]`을 적어주면 서로 섞이지 않고 구분되어 저장됩니다.
```bash
# 본계정 예시
python endfield_tracker_v1.1.py "11111" "본계"

# 부계정 예시
python endfield_tracker_v1.1.py "2222" "부계정"
```

### 2. 깔끔한 웹 뷰어(GUI)로 기록 보기
가챠 데이터를 모두 추출했다면, 아래 명령어를 실행하여 웹 브라우저에서 기록을 확인할 수 있습니다.
```bash
python viewer_web.py
```
- 서버가 켜지면 자동으로 브라우저 창이 열리며 계정별, 배너별 조회가 가능합니다.
- 콘솔에서 종합 평가를 보는 것도 여전히 가능하며, 전체 상세 기록은 `endfield_gacha_history_all.csv` 파일로 자동 추출됩니다.

## [주의사항]
- 본 프로그램은 비공식 도구입니다.
- 추출된 인증 토큰은 오직 게임의 공식 서버(ef-webview.gryphline.com)에서 데이터를 불러오기 위해서만 전송되며, 어떠한 외부 서버로도 유출되거나 로컬에 저장되지 않습니다.
- 본 도구의 사용으로 인해 발생하는 계정 관련 불이익에 대해서는 개발자가 책임지지 않습니다.
